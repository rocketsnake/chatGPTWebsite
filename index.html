Instructions

Step 1

  • Go to ChatGPT and create a free account (if you don't have one already).
  • Enter the following text into the prompt box and press enter:

"I need a prompt for creating an amazing website design for a __________ business"

  • Replace the "__________" part with the type of business you're building a website for. For example, if you're building a camping website, replace it with "camping".

Step 2

  • After ChatGPT gives you the prompt, copy that prompt and paste it into the new prompt box in your AI image generator , and hit enter.

IMPORTANT NOTE

Make sure you select the number of images you want it to produce: I recommend 4.

Use the prompt enhancer for getting 3 or 4 suggestions from the AI and select one you like.

Choose landscape 16:9 (landscape) aspect ratio, as it's the best for a website layout size.

Step 3

  • Pick an image that represents the site you want to create and download it
  • Go back to ChatGPT and click the plus button
  • Then type the prompt below into the chat box and press enter (or send)

Write the code for me to build a website using HTML, CSS, and JavaScript, if necessary, based on this image. Use pexels.com to insert any appropriate images in the site where needed.

  • Open a new text file and paste in the code. Then save the file as 'index.html'
